# Recall Web

> **Knowledge should be remembered, not merely stored.**

Frontend application for **Recall** — an AI-powered Personal Knowledge Management (PKM) platform built with React, Vite, TailwindCSS, and Lucide icons.

## Features (Sprint 1)

- **Authentication**: Modern Glassmorphism Login & Registration UI.
- **JWT Integration**: Automatic Bearer token persistence & request interceptor via Axios.
- **Protected Routing**: React Router DOM route guards.
- **PKM Aesthetic**: Customized dark theme with gradients and responsive glass panels.

## Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: TailwindCSS v4
- **Icons**: Lucide React
- **HTTP Client**: Axios
- **Routing**: React Router DOM v7

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The application will run at `http://localhost:3000`.
